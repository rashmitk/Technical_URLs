Q :To avoid typing username/password everytime you push your code to remote repository, do following:

$ git config credential.helper store
$ git push origin master
Username: <type your username>
Password: <type your password>

Check that above process should have created '.git-credentials' file with below content : 


https://rashmitk:4dm1n5tl_123@github.com

Now you dont need to supply username/password everytime you push your code to remote repo.
